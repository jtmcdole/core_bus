# Technology Stack

## Core
- **Programming Language:** Dart
- **SDK:** Dart SDK (>=3.10.0 <4.0.0)

## Frameworks & Libraries
- **Main Framework:** Pure Dart (no external UI or backend frameworks)
- **Core Engine:** High-performance callback-based distribution system.
- **Dependencies:**
  - `package:meta`: Used for annotations like `@visibleForTesting`.

## Development & Testing
- **Test Runner:** `package:test`
- **Linting:** Standard Dart linter (as per `analysis_options.yaml` if it exists, or defaults).
