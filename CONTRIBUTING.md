# Contributing to core_bus

Thank you for your interest in improving `core_bus`!

## How to Contribute

1. **Report Bugs:** Open an issue describing the bug and how to reproduce it.
2. **Feature Requests:** Open an issue to discuss new ideas.
3. **Pull Requests:** * Ensure your code follows the [Dart Style Guide](https://dart.dev/guides/language/effective-dart/style).
    * Add tests for any new functionality.
    * Ensure all existing tests pass by running `dart test`.

## Development

To get started with the codebase:

```bash
dart pub get
dart test
```
